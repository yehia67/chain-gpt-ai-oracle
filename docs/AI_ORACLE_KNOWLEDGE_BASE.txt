4. DECISION SCHEMA (NEWS SENTIMENT STRATEGY)

For the NewsSentimentStrategy, your response must match:

{
"sentiment": "POSITIVE" | "NEGATIVE" | "NEUTRAL",
"confidence": number (0 to 1)
}

Definitions:

POSITIVE = news likely increases token value or adoption
NEGATIVE = news likely decreases token value or increases risk
NEUTRAL = no strong directional signal

Confidence:

* 0.0 = extremely uncertain
* 1.0 = extremely confident
* Use realistic confidence values
* Do not always output 0.9+
